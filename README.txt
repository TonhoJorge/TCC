RESUMO
A presente pesquisa propõe o desenvolvimento de uma funcionalidade de cupons de desconto no aplicativo Strava, utilizando linguagens de programação como JavaScript, HTML e CSS. O modelo de negócio adota abordagens B2B e B2C, com parcerias estratégicas com empresas de artigos esportivos e oferta de descontos aos usuários. O estudo compreendeu uma pesquisa bibliográfica exploratória para embasar a proposta, o uso da API do Strava para integração (limitada a parceiros) e a apresentação de um protótipo de alta fidelidade da interface gráfica. A proposta também contemplou a criação de um banco de dados para suportar a funcionalidade. 

Palavras-chave: Strava, Aplicativo de Atividade Física, Modelo de Negócio Publicitário, Aplicativo com Recompensa, Aplicativo de Promoção de Saúde


ABSTRACT
This research proposes the development of a discount coupon functionality in the Strava application, using programming languages such as JavaScript, HTML, and CSS. The business model adopts B2B and B2C approaches, with strategic partnerships with sporting goods companies and offering discounts to users. The study included an exploratory literature review to support the proposal, the use of the Strava API for integration (limited to partners), and the presentation of a high-fidelity prototype of the graphical interface. The proposal also included the creation of a database to support the functionality.

Keywords: Strava, Fitness Application, Advertising Business Model, Reward-based Application, Health Promotion Application.
